# OTP Input — Machine Coding Cheat Sheet

## State

- `inputArray` → array (OTP digits)

## Refs

- `inputArrayRef` → array of input DOM refs

## Core Logic

- store one digit per input
- sanitize input to last numeric character
- auto-move focus forward
- auto-submit when OTP length is complete

## Handlers

- change input value
- move focus on input / backspace
- set cursor position on click

## Render

- map inputs based on OTP length

## Cleanup

- none
